One cycle. Two budgets. No ceremony.

Built like a tiny tool

Free phase switching

Dual progress rings

No notification permissions

Phase changes use Web Audio tones instead of OS notifications or background services.

Frontend-owned logic

Quick answers

Ketchup in plain facts.

What is Ketchup?
An open-source desktop Pomodoro timer built with Tauri v2, React, and TypeScript.
What makes it different?
Each cycle has both a work budget and a break budget, and you can switch between them freely.
Where does the timer logic live?
The state machine, persistence, and UI live in the React frontend; Rust only hosts Tauri plugins.
How is it distributed?
The landing page links directly to the latest Windows, macOS, and Linux release assets.
