The unusual bit

One cycle. Two budgets. No ceremony.

Every Pomodoro cycle allocates both a work budget and a break budget up front. You decide when to switch. Each side counts down independently. When both run out, the cycle completes.

Quick answers

Ketchup in plain facts.

What is Ketchup?
An open-source desktop Pomodoro timer built with Tauri v2, React, and TypeScript.
What makes it different?
Each cycle has both a work budget and a break budget, and you can switch between them freely at any moment.
Where does the timer logic live?
The state machine, persistence, and UI live in the React frontend. Rust only hosts Tauri plugins.
How is it distributed?
The landing page links directly to the latest Windows, macOS, and Linux release assets—no GitHub account needed.
